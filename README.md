# Ptable
Future Style Periodic Table is a native iOS periodic table app for iPhone and iPad. Unlike traditional static charts, this app features a dark Cyberpunk aesthetic with neon glow effects, Glassmorphism, and fluid native animations — redefining how chemical elements are visualized.
The app covers all 118 elements with comprehensive data including atomic number, relative atomic mass, electron configuration, isotopes, and valence. It also supports multi-dimensional heatmap visualization and 3D atomic structure simulation — the perfect tool for learning chemistry and exploring the elements.
✨ Highlight: Tap any element to enter an immersive detail view. Drag and rotate the 3D atomic model with touch gestures to experience the beauty of electron orbitals. Multi-language support available for users worldwide.

✨ Features
🎨 Immersive Visual Experience

Cyberpunk Aesthetics: Dark background with dynamic particle effects rendered via Metal, creating a deeply atmospheric, high-tech vibe.
Neon Category Highlights: 10 element categories each with a unique color scheme — borders glow and flow on touch, fully optimized for ProMotion displays.
Universal Device Support: From iPad Pro to iPhone SE, elegant adaptive layouts with free portrait/landscape switching.
Multi-language Support: Switch interface language with one tap, deeply integrated with system language settings.

📊 Multi-Dimensional Data Visualization

Heatmap Mode: Instantly switch between atomic radius, electronegativity, ionization energy, melting point, and boiling point distributions — rendered natively with Swift Charts.
Category Filter: Tap any legend item to highlight a specific category (alkali metals, noble gases, lanthanides/actinides, and more).
Live Search: Quickly locate elements by symbol, name, or atomic number — with Spotlight Search integration.

🔬 3D Atomic Structure Simulation

SceneKit 3D Orbital Model: Real-time rendering of electron shells and rotating electrons based on electron configuration algorithms, fully GPU-accelerated.
Native Gesture Interaction: Pinch to zoom, two-finger rotate — observe atomic structure from any angle at a silky-smooth 120fps.
Full-Screen Immersive Mode: Tap the model to enter full-screen view; pinch to zoom in on electron shell details.
Detailed Data Cards: Electron configuration notation, shell electron counts, isotope stability, common valences, physical & chemical properties, element history, and visual gallery.


🎯 Who Is It For?

Middle School Students: New to the periodic table? Neon color coding and category highlights make it easy to distinguish metals, nonmetals, and more. 3D atomic models bring abstract structures to life — making memorization actually fun.
High School Students: The ultimate study companion. Heatmap mode visualizes periodic trends at a glance — electronegativity, ionization energy, valence patterns — so you understand instead of memorize.
University Students & Researchers: Look up atomic mass, electron configurations, isotope data, and more on the go. A pocket-sized replacement for bulky chemistry references — ready in the lab or library.
Educators: Demonstrate periodic trends in class with live category filtering and heatmap switching — turning abstract concepts into something students can actually see.
Chemistry Enthusiasts: Curious about any element? Just search and dive in. The immersive experience makes exploring the periodic table a pleasure in itself.
